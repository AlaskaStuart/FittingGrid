# FittingGridAlgorithm

这是一个关于自动生成矿山勘探孔网格的算法。

## 算法描述

基本思路是：

1. 平移所有钻孔坐标点至原点
2. 计算霍夫空间下曲线的交点
3. 对交点的$\theta$值应用核密度估计，找到最大的两个$\theta$值
4. 计算$\theta$值对应的基
5. 以这两个基为轴，将平移后的坐标点进行线性变换
6. 分别计算两个轴上线性变换后点的核密度，将核密度峰值存入列表
7. 计算所有平移变换点与核密度峰值之间的距离，取距离最小者在列表中的位置作为平移变换点在网格中的位置
8. 可能存在重叠情况，因此采用“先到先得”和“就近移动原则”，对重复值进行处理。

## 需要的数据格式

- csv 文件，包含钻孔坐标点的坐标信息，格式为：`drill_name`, `x`, `y`，三列

## 存在的问题

- 边界限制问题，可能触发 `IndexOutOfRange` 异常。
- 核密度估计采用高斯核密度估计，目前的 `bandwith` 为经验值。
- 算法未考虑勘探孔的坐标点存在完全垂直的情况，可能会导致出零错误。
